はじめに

このページは、HTML、CSS、JavaScriptの基本を学ぶための初心者向けサイトです。
コードを実際に編集して、どのように変化するかを確認しましょう。

Webページの3要素

  • HTML: ページの構造（骨組み）
  • CSS: ページの見た目（装飾）
  • JavaScript: ページに動き（機能）

HTMLの基本

HTMLはWebページの構造を作る言語です。タグを使って内容を定義します。

HTMLの基本構造

<!DOCTYPE html>
<html>
<head>
  <title>ページのタイトル</title>
</head>
<body>
  <h1>見出し</h1>
  <p>段落のテキスト</p>
</body>
</html>

CSSの基本

CSSはHTML要素の見た目を整えるための言語です。

CSSの書き方

1. インラインスタイル

<p style="color: blue;">青い文字</p>

2. 内部CSS（styleタグ内）

<style>
p { color: blue; }
</style>

3. 外部CSS（推奨）

<link rel="stylesheet" href="style.css">

JavaScriptの基本

JavaScriptはWebページに動きや対話性を追加するための言語です。

JavaScriptの書き方

1. インラインJavaScript

<button onclick="alert('こんにちは！')">クリック</button>

2. 内部JavaScript

<script>
function sayHello() {
  alert('こんにちは！');
}
</script>

3. 外部JavaScript（推奨）

<script src="script.js"></script>
