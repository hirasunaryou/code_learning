はじめに

このページは、完成済みの style.css を事例として、CSS の基本スタイル、デザインの工夫、そしてレスポンシブ対応などを詳しく解説する教材です。
読み物として各要素の役割や設定意図を理解し、実際にスタイルを変更してその効果を体験してください。

1. 基本のレイアウトと基本スタイル

1-1. body セレクタによる基本スタイル

body {
  font-family: 'Arial', 'Helvetica', sans-serif;
  line-height: 1.6;
  margin: 0;
  padding: 0;
  background-color: #f8f9fa;
  color: #333;
  transition: background-color 0.3s, color 0.3s;
}

このコードは、全体のフォント、行間、余白、背景色、文字色を設定しています。
transition を使用することで、背景色や文字色の変更を滑らかに演出できます。

実験してみよう！

  • 背景色や文字色、フォントの設定値を変更して、ページ全体の印象がどのように変わるか確認する。

1-2. ヘッダーとナビゲーションのスタイル

header {
  background-color: #4a6fa5;
  color: white;
  text-align: center;
  padding: 1.5rem;
}

header h1 {
  margin: 0;
  font-size: 2rem;
}

nav {
  display: flex;
  justify-content: center;
  background-color: #3a5a80;
  padding: 0.5rem;
}

nav a {
  color: white;
  text-decoration: none;
  padding: 0.5rem 1rem;
  margin: 0 0.5rem;
  border-radius: 4px;
  transition: background-color 0.3s;
}

nav a:hover {
  background-color: #2c4562;
}

ここでは、ヘッダー部分とナビゲーションバーのスタイルを定義しています。
背景色、テキスト色、パディング、レイアウト（フレックスボックスを使用）などを調整し、ユーザーが見やすく操作しやすいデザインに仕上げています。

実験してみよう！

  • ヘッダーやナビゲーションの背景色や文字色を変更して、デザインの印象がどう変わるか試す。

2. セクションと見出しのスタイル

2-1. section のスタイル

section {
  background-color: white;
  border-radius: 8px;
  box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1);
  margin-bottom: 2rem;
  padding: 1.5rem;
}

このスタイルは、各セクションの背景色、角の丸み、影、余白、パディングを設定しています。
セクションごとに区切られたカードのような見た目にすることで、内容が整理され見やすくなります。

実験してみよう！

  • box-shadow や border-radius の値を変えて、セクションの印象がどのように変わるかを確認する。

2-2. h2, h3 のスタイル

h2 {
  color: #4a6fa5;
  border-bottom: 2px solid #4a6fa5;
  padding-bottom: 0.5rem;
  margin-top: 0;
}

h3 {
  color: #5a7db6;
  margin-top: 1.5rem;
}

見出し要素 (h2, h3) の色、ボーダー、パディング、マージンを設定することで、タイトル部分が際立ち、全体のデザインが引き締まります。

実験してみよう！

  • h2 と h3 のカラーや余白、ボーダーを変更して、どのような視覚効果が得られるかを確認する。

3. 補助的なスタイルとレスポンシブ対応

3-1. コードブロックと例示ボックスのスタイル

pre {
  background-color: #f5f5f5;
  border: 1px solid #ddd;
  border-radius: 4px;
  padding: 1rem;
  overflow-x: auto;
  font-family: 'Courier New', Courier, monospace;
}

code {
  font-family: 'Courier New', Courier, monospace;
  background-color: #f5f5f5;
  padding: 0.2rem 0.4rem;
  border-radius: 3px;
}

このスタイルにより、サンプルコードが読みやすく表示されます。
例示ボックス（.example-box）は、補足情報やサンプルを際立たせるために使用されます。

実験してみよう！

  • pre や code のスタイルを調整して、コードの表示がどのように変わるかを確認する。

3-2. レスポンシブ対応の工夫

@media (max-width: 768px) {
  .two-column {
      flex-direction: column;
  }

  nav {
      flex-direction: column;
      align-items: center;
  }

  nav a {
      margin: 0.2rem 0;
      width: 80%;
      text-align: center;
  }
}

このメディアクエリは、画面幅が狭い場合にレイアウトを縦並びに変更するなど、レスポンシブなデザインを実現するための設定です。

実験してみよう！

  • ウィンドウサイズを変更し、スタイルの変化を確認する。

よくある質問（FAQ）とデバッグのヒント

初心者がよく直面する疑問やエラー、またその解決策を以下にまとめました。

  • エラー例: プロパティ名のスペルミスやセミコロンの抜けなどにより、スタイルが適用されない場合があります。
  • デバッグ方法: ブラウザの開発者ツールでスタイルの適用状況を確認し、問題箇所を特定してください。
