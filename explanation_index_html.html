はじめに

このページは、完成済みの index.html を事例として、HTML の基本構造、文法、書き方の工夫などを詳しく解説する教材です。
読み物として各要素の役割や意図を理解し、実際にコードを編集してその効果を体験してください。

1. HTMLの基本構造とは？

1-1. ドキュメントの最初の宣言

<!DOCTYPE html>

この宣言は、文書が HTML5 で書かれていることをブラウザに伝えます。
最新の仕様に基づいて正しく表示するために必須です。

実験してみよう！

  • この宣言を一時的に削除し、ブラウザの表示がどう変わるか確認してみましょう。

1-2. <html>タグと lang 属性

<html lang="ja">

<html> タグは文書全体のルート要素です。
lang="ja" 属性は、このページが日本語であることを示し、検索エンジンや支援技術の動作を最適化します。

実験してみよう！

  • lang 属性の値を "en" に変更して、どのような影響があるかを確認してみる。

1-3. <head> セクションの役割と構成

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>初心者のためのHTML・CSS・JavaScript学習ページ</title>
  <link rel="stylesheet" href="style.css">
</head>

<head> 内では、文書全体の設定（文字コード、ビューポート、タイトル、外部リソースの読み込み）を行います。
これにより、正しく表示され、様々なデバイスに対応できるようになります。

実験してみよう！

  • <title> の内容を変更して、ブラウザタブに反映されるタイトルを確認する。
  • <meta charset> の値を変えて、文字化けが起こるかどうかを実験してみる。

1-4. <body> セクションの構成と各コンテンツ

<body>
  <header>
    <h1>初心者のためのHTML・CSS・JavaScript学習ページ</h1>
  </header>
  <nav>
    <a href="#intro">はじめに</a>
    <a href="#html">HTML</a>
    <a href="#css">CSS</a>
    <a href="#js">JavaScript</a>
    <a href="#playground">実験場</a>
  </nav>
  <main>
    <section id="intro">
      <h2>はじめに</h2>
      <p>このページは...</p>
    </section>
    ...
  </main>
  <footer>
    <p>© 2025 HTML・CSS・JavaScript学習サイト</p>
  </footer>
</body>

<body> には、実際にユーザーに表示されるコンテンツが含まれます。
ヘッダー、ナビゲーション、メインコンテンツ、フッターという構成で、各部分の役割を理解することで、全体の仕組みが把握できます。

実験してみよう！

  • ヘッダー内の見出しテキストを変更して、ページ全体の印象がどのように変わるか確認する。
  • ナビゲーションリンクを追加・削除して、ページ内の移動がどのように動作するか実験する。

2. スタイルとの連携とその効果

2-1. 外部CSSの読み込み

<link rel="stylesheet" href="style.css">

このタグにより、style.css 内のスタイルがページに適用されます。
HTMLとスタイルを分離することで、コードの再利用性と管理がしやすくなります。

実験してみよう！

  • style.css の一部をコメントアウトし、ページの見た目がどう変わるかを確認する。

2-2. CSSがページに与える視覚的効果

CSS は、文字のフォント、色、背景、レイアウト、ボタンのデザインなど、ページの視覚的な印象を大きく左右します。
たとえば、transition プロパティを使うと、色の変化が滑らかになります。 

body {
  font-family: 'Arial', 'Helvetica', sans-serif;
  line-height: 1.6;
  margin: 0;
  padding: 0;
  background-color: #f8f9fa;
  color: #333;
  transition: background-color 0.3s, color 0.3s;
}

実験してみよう！

  • 背景色や文字色、フォントサイズを変更して、どのような視覚効果が得られるか確認する。

3. 動的機能とインタラクションの実現

3-1. 外部JavaScriptの読み込み

<script src="script.js"></script>

このタグにより、script.js 内の JavaScript が読み込まれ、ページの動的な動作やインタラクションが実現されます。
JavaScript を利用することで、ユーザーの操作に応じた動作やコンテンツの更新が可能になります。

実験してみよう！

  • このタグを一時的に削除して、ページの動作にどのような影響があるか確認する。

3-2. ページ動作を支える基本的なJavaScriptコード

// ページ読み込み後の初期化処理
document.addEventListener('DOMContentLoaded', () => {
  hljs.highlightAll();
  initCodeMirrorEditors();
});

このコードは、ページが完全に読み込まれた後に実行される初期化処理です。
hljs.highlightAll() によりコードがシンタックスハイライトされ、initCodeMirrorEditors() で各テキストエリアが高機能なエディタに変換されます。

実験してみよう！

  • この初期化処理に変更を加え、読み込み完了時にコンソールへメッセージを出力するようにして動作を確認する。

よくある質問（FAQ）とデバッグのヒント

初心者が直面しやすいエラーや疑問点、その解決策を以下にまとめました。

  • エラー例: "Unexpected token" → JavaScript の文法ミスが原因の場合があります。コードの記述ミスを注意深く確認してください。
  • デバッグ方法: ブラウザのデベロッパーツールを利用し、console.log() で変数の状態や処理の流れを確認するのがおすすめです。
